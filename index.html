Pourquoi Kotlin?

En 2011 Jetbrains annonçait le développement du langage Kotlin qui est rapidement devenu le langage officiel du premier système d'exploitation mobile. Pour comprendre l'attrait pour Kotlin, il faut d'abord comprendre le rôle de Java dans le paysage du développement logiciel. Les deux langages sont étroitement liés, parce que le code Kotlin est le plus souvent écrit pour la machine virtuelle Java, ils sont interopérables. De plus Kotlin est un langage multipateforme généraliste qui peut être utilisé pour écrire des applications MacOs et Windows natives, des applications JavaScript, ou encore des applications Android.
